音声ファイルのアップロード

複数ファイルを選択すると、ファイル1つ＝話者1人として話者分離されます。
単一ファイルの場合は全員ミックス音声として処理されます（話者分離不可）。
話者名はファイル選択後に編集欄が表示されます（ファイル名から自動推定・編集可）。
制約: 1ファイルあたり 最大100MB / 最大60分 ・ 対応拡張子: .m4a / .mp4 / .wav / .mp3 / .webm / .ogg / .flac
上のボタンが使えないブラウザ（Safari等）は下のフィールドから選択してください
話者名（ファイル名から自動推定。必要なら編集してください）

録音音声を追加

既存プロジェクトに音声ファイルを追加します。ファイル1つ＝話者1人として登録され、追加後は録音リストの末尾に並びます。
話者名はファイル名から自動推定・編集可能。
制約: 1ファイル 最大100MB ・ 対応: .m4a / .mp4 / .wav / .mp3 / .webm / .ogg / .flac
話者名（ファイル別に編集可）
