入力ソース

3 種類の入力を同じパイプラインへ

対面・オンライン・既存音声のどれでも、同じ4カラム比較ビュー・要約・Excel エクスポートまで一本道。

SOURCE 1

モバイル PWA

  • iPhone Safari からホーム画面追加で起動
  • 20分自動 pause・残時間表示・STT 無反応バナー
  • IndexedDB 退避で電波切れにも強い
  • 録音中の話者ボタンが話者確定の決め手

SOURCE 2

Zoom Cloud Recording

  • 管理画面の一覧から1件ずつ取り込み（Pull 型）
  • 参加者別音声の分離トラックで話者100%確定
  • Webhook は通知のみ。意図しない自動 DL なし
  • Server-to-Server OAuth で自アカウント録画を取得

SOURCE 3

任意音声ファイル

  • m4a / mp4 / wav / mp3 / webm / ogg / flac 対応
  • 複数選択でファイル=話者として分離取り込み
  • 1ファイル最大100MB・最大60分（クライアント検証）
  • ICレコーダー・スマホ・他社会議システム録画も同じ流れで処理
