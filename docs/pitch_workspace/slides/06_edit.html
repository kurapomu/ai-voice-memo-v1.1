編集と運用

人手による補正を低コストで

完全自動を目指さず、人間が必要な箇所だけ素早く直せる UI を提供する。

EDIT 01

行ごとの話者修正

  • LLM 最終版の話者ラベルをクリックでドロップダウン化
  • 参加者リストから1クリックで差し替え
  • 再マージ不要・Excel 出力にも即同期

EDIT 02

録音音声の追加・順序入替

  • ↑↓ ボタンで録音の順序を即時入替
  • 追加収録した音声を後から差し込み可能
  • 古い解析結果には警告バナーで再実行を促す

EDIT 03

マージパラメータ調整

  • チャンクサイズ・並列度・主軸選定アルゴリズムを画面で変更
  • 重み（Web Speech : Run1 : Run2 = 10合計）でテキスト採用バランスを制御
  • 運用デフォルトは Whisper 主軸（1:7:2）
