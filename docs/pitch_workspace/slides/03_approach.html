解決アプローチ

二段階方式 - 人間 × AI の役割分担

録音中に人間が「誰が話したか」を担い、後処理で AI が「何を話したか」を担う。最後に LLM が両者を統合する。

REAL-TIME LAYER

リアルタイム層記録時にだけ可能な情報を取り逃さない

  • Web Speech API で録音中に即時文字起こし
  • 話者ボタン押下で話者を確定（人手・最高信頼）
  • 20分自動 pause、STT 無反応バナーで安全運用

POST-PROCESS LAYER

後処理層大量データを高品質テキストに整える

  • OpenAI Whisper でテキスト精度を確保（Run1）
  • Deepgram Nova-3 で話者分離と分節（Run2）
  • Run1 と Run2 のエンジン・モデルは個別に選択可能
